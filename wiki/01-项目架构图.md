## 3. 数据流架构

```mermaid
sequenceDiagram
    participant User as 用户/前端
    participant API as FastAPI
    participant Bot as TradingBot
    participant Selector as StrategySelector
    participant Strategy as 具体策略
    participant Aster as AsterClient
    participant Exchange as AsterDEX

    %% 启动流程
    User->>API: POST /bot/start
    API->>Bot: 初始化TradingBot
    Bot->>Aster: 创建AsterFuturesClient
    Aster->>Exchange: 获取账户信息
    Exchange-->>Aster: 返回账户数据
    Aster-->>Bot: 返回初始资金
    Bot->>Bot: 初始化RiskManager
    Bot->>Bot: 初始化策略列表
    Bot->>Selector: 初始化StrategySelector
    Bot-->>API: 启动成功
    API-->>User: 返回成功状态

    %% 交易循环
    loop 每60秒迭代
        Bot->>Aster: 获取K线数据
        Aster->>Exchange: GET /fapi/v1/klines
        Exchange-->>Aster: 返回OHLCV数据
        Aster-->>Bot: 返回DataFrame
        
        Bot->>Aster: 获取订单簿
        Aster->>Exchange: GET /fapi/v1/depth
        Exchange-->>Aster: 返回订单簿
        Aster-->>Bot: 返回bids/asks
        
        Bot->>Selector: 分析市场并选择策略
        Selector->>MarketRegime: 检测市场状态
        MarketRegime-->>Selector: 返回regime + confidence
        
        Selector->>Strategy: 调用策略分析
        Strategy->>Strategy: 计算技术指标
        Strategy-->>Selector: 返回交易信号
        
        alt 有交易信号
            Selector-->>Bot: 返回最佳策略信号
            Bot->>RiskMgr: 检查风险限制
            RiskMgr-->>Bot: 允许/拒绝开仓
            
            alt 允许开仓
                Bot->>Aster: 创建订单
                Aster->>Exchange: POST /fapi/v3/order
                Exchange-->>Aster: 返回订单确认
                Aster-->>Bot: 返回orderId
                Bot->>RiskMgr: 记录持仓
                Bot->>Aster: 设置止损单
                Bot->>Aster: 设置止盈单
            end
        end
        
        Bot->>Aster: 检查持仓状态
        Aster->>Exchange: GET /fapi/v3/positionRisk
        Exchange-->>Aster: 返回持仓数据
        Aster-->>Bot: 更新持仓PnL
    end

    %% 查询流程
    User->>API: GET /dashboard/summary
    API->>Bot: 获取统计数据
    Bot->>Aster: 获取实时账户数据
    Aster->>Exchange: GET /fapi/v3/account
    Exchange-->>Aster: 返回账户信息
    Aster-->>Bot: 返回equity/PnL
    Bot-->>API: 返回完整统计
    API-->>User: 返回JSON数据
```
